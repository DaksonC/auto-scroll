# auto-scroll
Extensão para Google Chrome que realiza o scroll automático. Com um clique inicia o scroll e com outro para o scroll. Na barra você muda a velocidade do scroll. Bem útil para leitura e para fazer aquela gravação marota do demo do site que acabara de criar.
